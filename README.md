# 🛡️ ShieldSMS AI

AI-Powered Scam & Spam SMS Detection System

## Overview

ShieldSMS AI is a lightweight AI-inspired security application that analyzes SMS messages and detects potential scams, phishing attempts, spam, and suspicious content.

The system evaluates messages using risk indicators such as:

* Suspicious links
* Urgency language
* Financial impersonation
* Scam keywords
* Suspicious phone numbers

and generates:

* Risk Score
* Scam Probability
* AI Confidence Score
* Risk Category
* Recommended Action
